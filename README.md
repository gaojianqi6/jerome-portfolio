# Jerome Portfolio Product Docs

This folder contains the product planning documents for Jerome Gao's personal portfolio site.

- [PRFAQ](./docs/PRFAQ.md): reverse-working narrative, launch story, and FAQ.
- [PRD](./docs/PRD.md): structured product requirements, information architecture, task flows, boundaries, and data model.
- [Design Spec](./docs/DESIGN_SPEC.md): visual direction, interaction model, design tokens, components, states, accessibility, and wireframes.
- [Technical Architecture](./docs/TECH_ARCHITECTURE.md): Next.js, React, CSS Modules, Base UI, lucide-react, i18n, content model, compatibility, testing, and upgrade strategy.
- [AI Project Guide](./docs/AI_PROJECT_GUIDE.md): how to use AI to build projects while avoiding generic, AI-generated product and visual patterns.

Working language: Chinese.
